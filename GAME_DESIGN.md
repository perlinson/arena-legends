# ⚔️ ARENA LEGENDS - 即时制炉石传说

## 游戏概述

> **Arena Legends** - 融合炉石传说的卡牌深度 + 皇室战争的即时策略

---

## 🎯 核心设计理念

### 炉石传说 (Hearthstone)
✅ **保留**:
- 卡牌关键词（战吼、冲锋、嘲讽、剧毒等）
- 法术机制
- 英雄技能
- 职业特色设计

### 皇室战争 (Clash Royale)
✅ **借鉴**:
- 实时对战（无需等待对手）
- 圣水自动恢复
- 战场空间感（放置位置）
- 快节奏（3-5分钟/局）

### 创新设计 (NEW!)
- **即时卡牌释放** - 拖拽施法，走位躲避
- **战场位移系统** - 随从可以移动
- **连携技** - 特定卡牌组合触发
- **终极技能** - 充能系统

## 🏗️ 核心机制设计

### 1. 圣水系统（实时版）

```javascript
// 每秒恢复 1 点圣水，最大 10 点
const MANA_SYSTEM = {
    maxMana: 10,
    manaRegenPerSecond: 1,
    initialMana: 2,  // 开局2费
    
    // 即时生效，无需等待
    isInstant: true
};
```

**关键区别**：
- 皇室战争：固定时间恢复，不能暂停
- Arena Legends：战斗时实时恢复，暂停时停止

### 2. 战场布局

```
    敌方英雄
   ┌─────────┐
   │  后排区  │ ← 法术、远程单位
   ├─────────┤
   │  中排区  │ ← 辅助、坦克
   ├─────────┤
   │  前排区  │ ← 冲锋、战士
   └─────────┘
    我方英雄
```

**位置机制**：
- 拖拽卡牌到战场任意位置
- 近战单位需靠近敌人才能攻击
- 远程单位有射程限制
- 某些卡牌只能放置特定区域

### 3. 随从系统

```typescript
interface Minion {
    // 基础属性
    attack: number;
    health: number;
    speed: number;      // 移动速度
    
    // 关键词
    keywords: ['charge' | 'taunt' | 'rush' | 'poison' | 'divineShield' | 'windfury'];
    
    // 即时制特有
    canMove: boolean;   // 是否可以移动
    attackRange: number; // 射程（近战=1，远攻=3-5）
    
    // 特殊能力
    abilities: SpecialAbility[];
}

interface SpecialAbility {
    trigger: 'onPlay' | 'onAttack' | 'onDeath' | 'onFriendlyDeath';
    effect: () => void;
}
```

### 4. 战斗流程（实时版）

```
战斗开始
    ↓
圣水自动恢复（1费/秒）
    ↓
玩家实时操作：
├─ 拖拽卡牌到战场
├─ 激活英雄技能
├─ 使用法术
└─ 指令随从移动/攻击
    ↓
目标：摧毁敌方英雄
    ↓
战斗胜利/失败
```

**与炉石的区别**：
| 炉石传说 | Arena Legends |
|---------|--------------|
| 回合制，轮流操作 | 实时，双方同时操作 |
| 随从原地不动 | 随从可以自由移动 |
| 等待对手响应 | 无需等待，即时生效 |
| 思考时间长 | 需要快速决策 |

```typescript
interface UltimateSkill {
    name: string;

    charges: 0;       // 当前层数

    name: '炎爆术',
```
